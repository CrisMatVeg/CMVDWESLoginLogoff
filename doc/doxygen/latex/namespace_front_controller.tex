\doxysubsection{Detailed Description}
Inicializa la sesión y establece la página por defecto

Este script sirve como punto de entrada de la aplicación. Se encarga de iniciar la sesión, establecer la página por defecto en caso de que no exista, y cargar el controlador correspondiente según la página actual.
