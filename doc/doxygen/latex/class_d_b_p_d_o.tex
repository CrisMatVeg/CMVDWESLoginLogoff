Ejecuta una consulta SQL en la base de datos y devuelve el primer resultado.

Este método utiliza PDO para conectarse a la base de datos utilizando los parámetros definidos en \doxylink{conf_d_b_p_d_o_8php}{conf\+DBPDO.\+php}. Permite pasar parámetros opcionales para consultas preparadas.
