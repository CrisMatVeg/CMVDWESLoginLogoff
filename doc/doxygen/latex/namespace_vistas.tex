Vista Login/\+\+Logoff

Este archivo es la plantilla principal de la interfaz de login y logoff. Incluye la estructura HTML, enlaces a hojas de estilo y scripts, definición de fuentes y variables CSS, y carga dinámica de la vista correspondiente según la página en curso en la sesión.

Dependencias:\+
\begin{DoxyItemize}
\item \$view\+[\$\+\_\+\+SESSION\+[\textquotesingle{}pagina\+En\+Curso\textquotesingle{}]\+]\+ → archivo PHP de vista que se carga dinámicamente
\end{DoxyItemize}

Funcionalidad:\+
\begin{DoxyItemize}
\item Define la estructura HTML5 con {\ttfamily \texorpdfstring{$<$}{<}header\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}main\texorpdfstring{$>$}{>}} y {\ttfamily \texorpdfstring{$<$}{<}footer\texorpdfstring{$>$}{>}}
\item Carga CSS externo y define estilos internos para el login/\+logoff
\item Muestra enlaces a recursos externos (Font Awesome, Duolingo, Github)
\end{DoxyItemize}
