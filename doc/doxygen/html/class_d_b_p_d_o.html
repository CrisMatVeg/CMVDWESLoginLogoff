ejecutarConsulta ( $sql,
$parametros = null )
static

Ejecuta una consulta SQL en la base de datos y devuelve el primer resultado.

Este método utiliza PDO para conectarse a la base de datos utilizando los parámetros definidos en confDBPDO.php. Permite pasar parámetros opcionales para consultas preparadas.

Parameters
string$sqlLa consulta SQL a ejecutar.
array | null$parametros(Opcional) Array de parámetros para la consulta preparada.
Returns
mixed Retorna el primer registro devuelto por la consulta como objeto/array, según fetch().
Exceptions
PDOExceptionLanza excepción si ocurre un error en la consulta o la conexión.
